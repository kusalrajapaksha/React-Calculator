# React Calculator

This is a simple calculator application built using React.js. It allows users to perform basic arithmetic operations such as addition, subtraction, multiplication, and division. Users can input digits, perform operations, clear the display, delete digits, and evaluate expressions.

## Features

- Addition, subtraction, multiplication, and division operations.
- Clear button to reset the calculator.
- Delete button to remove the last digit entered.
- Evaluation of expressions to get the result.
- Responsive design suitable for various screen sizes.

## How to Use

1. Clone the repository to your local machine.
2. Navigate to the project directory.
3. Install dependencies by running `npm install`.
4. Start the development server with `npm start`.

## Usage

- Click on digits (0-9) to input numbers.
- Use a decimal point (.) for floating-point numbers.
- Select an operation (+, -, *, /) to perform calculations.
- Press the "AC" button to clear the display and reset the calculator.
- Press the "DEL" button to delete the last digit entered.
- Click on "=" to evaluate the expression and display the result.

## Technologies Used

- React.js
- JavaScript
- HTML5
- CSS3

## File Structure

- `App.js`: The main component containing the calculator logic.
- `styles.css`: Stylesheet for the calculator layout and design.
- `components/`: Directory containing components used in the calculator.
  - `DigitButton.js`: Component for digit buttons (0-9).
  - `OperationButton.js`: Component for operation buttons (+, -, *, /).
